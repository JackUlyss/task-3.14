## git show
Выводит полный список изменений указанного коммита

```
git show b7cffb98fc2689f295ae3fd59bbc971e1eff3351
```
Коммит указывается при помощи хеша, который можно сократить до первых 6 символов: 

```
git show b7cffb
```
